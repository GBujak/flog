# flog (fast log)

```
Usage: flog <COMMAND>

Commands:
  add-dir            Add directory to git branch sources
  rm-dir             Remove directory from git branch sources
  set-branch-format  Set branch format. Branch name will be split on separator and index-th element will be used for worklog tag
  set-tag-prefix     Set the prefix that will be used for every worklog tag
  print-config       Show current configuration
  log                Create new work log
  help               Print this message or the help of the given subcommand(s)

Options:
  -h, --help  Print help information
```
